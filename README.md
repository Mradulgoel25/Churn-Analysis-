# Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn behavior in a telecom company using Python and Exploratory Data Analysis (EDA). The goal is to identify factors that influence customer retention and understand patterns associated with customer churn.

## 🎯 Objectives
- Analyze customer demographics and service usage.
- Identify factors contributing to customer churn.
- Visualize customer behavior using charts and plots.
- Generate actionable insights for improving customer retention.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset Features
The dataset contains customer information such as:
- Gender
- Senior Citizen Status
- Tenure
- Contract Type
- Phone Service
- Internet Service
- Online Security
- Online Backup
- Device Protection
- Tech Support
- Streaming TV
- Streaming Movies
- Payment Method
- Churn Status

## 📊 Analysis Performed
### 1. Data Cleaning
- Replaced blank values in `TotalCharges`.
- Converted data types where required.
- Checked for missing and duplicate values.

### 2. Exploratory Data Analysis
- Customer churn distribution.
- Churn by gender.
- Churn by tenure.
- Churn by contract type.
- Churn by payment method.
- Service adoption analysis.

### 3. Service Usage Analysis
The following services were analyzed:
- Phone Service
- Multiple Lines
- Internet Service
- Online Security
- Online Backup
- Device Protection
- Tech Support
- Streaming TV
- Streaming Movies

## 📈 Key Insights
- Customers with month-to-month contracts show higher churn rates.
- Long-tenure customers are less likely to churn.
- Value-added services such as Online Security and Tech Support have lower adoption rates.
- Fiber optic internet users show significant churn behavior.
- Contract type plays a major role in customer retention.
